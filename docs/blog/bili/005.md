### pre

- [sys V ABI](https://wiki.osdev.org/System_V_ABI)

    - 调用约定
        - 传参 : rdi -> rsi -> rdx -> rcx -> r8 -> r9 -> 栈

        - 被调用方保存 (实现方):

            - `rbp` - `r12`
            - `r13` - `r14` - `r15`

64 位下重载段寄存器?

    - -> far jmp (protected mode)
    - -> iretq / 中断返回
